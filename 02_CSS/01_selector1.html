선택자

1. 전체 선택자: *

지금부터 스타일시트를 이용한 문서 표현 방법을 배워요

2. 태그 선택자: 태그 이름

태그 선택자는 태그이름이 선택자로 사용되는 유형

태그 선택자와 같은 이름의 모든 태그에 css3 스타일 시트 내용 적용

여러 선택자에 동일한 스타일 속성을 지정할 경우 선택자들을 쉼표로 구분

3. 아이디 선택자: #id명

요소의 id속성으로 지정된 이름

  • HTML5
  • css3
  • Javascript
  • 웹프로그래밍
  • 자료구조

4. 클래스 선택자: .클래스명

요소의 class속성으로 지정된 영역에 스타일을 지정

변수
프로그래밍 언어에서 데이터값을 저장하는 메모리 공간을 의미
연산자
파이썬의 연산자는 기본적으로 산술, 관계, 논리, 비트 연산자가 있다.
id 속성
각 태그를 유일하게 구분
동일한 id속성을 갖지 않는다.
id 선택자는 여러 태그 중 특정 태그에만 css스타일을 적용할 때 주로 사용
class 속성
여러 태그를 하나의 그룹으로 묶어서 동일한 스타일을 적용하기 위해 사용
클래스 속성을 여러 태그에 지정할 수 있다.

5. 자식(child)선택자와 후손(descendent)선택자

계층구조로 되어 있는 HTML 태그 관계에서 부모와 자식 관계는 현재 태그를 포함하는 태그는 부모태그, 현재 태그는 자식태그가 된다.

후손태그는 자식을 포함하여 하위에 포함된 모든 태그들

  • 자식 선택자: 부모 선택자 > 자식 선택자
  • 후손 선택자: 선조 선택자 후손 선택자1, 선조 선택자 후손 선택자2,.....
