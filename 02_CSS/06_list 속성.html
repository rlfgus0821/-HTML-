리스트 모양을 꾸미기 위한 속성

  • list-style: 마커타입,마커이미지, 위치 지정
  • list-style-type: 아이템의 마커 타입 지정
    • /* disc | circle | square | decimal | alpha | lower-roman | upper-roman | none */
  • list-style-image: 아이템 마커의 이미지 지정
    • /* list-style-image : url('media/apple.png') */
  • list-tyle-position: 아이템 마커 출력 위치 지정
    • /* inside, ouside */
