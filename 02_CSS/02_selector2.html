6. 가상클래스(pseudo-class)

어떤 조건이나 상황에서 스타일을 적용하도록 하는 선택자

40개 이상의 많은 가상 클래스 선택자가 있다.

  • 마우스
    • :hover => 마우스가 올라갈때 스타일 적용
    • :active => 마우스로 누르고 있는 상황에서 스타일 적용
  • 폼(form) 요소
    • :focus => 폼 요소가 키보드나 마우스 클릭으로 포커스를 받을 때 스타일 적용
    • :checked => 체크상태의 input 태그에 스타일 적용
    • :enabled => 사용가능한 input 태그에 스타일 적용
    • :disabled => 사용 불가능한 input 태그에 스타일 적용
  • 하이퍼링크
    • :link => 방문하지 않은 링크에 스타일 적용
    • :visited => 방문한 링크에 스타일 적용
  • 블록: p, div와 같은 블록형 태그
    • :first-letter => 블록형 태그의 첫 글자에 스타일 적용
    • first-line => 블록형 태그의 첫 줄에 스타일 적용
  • 구조
    • :nth-child(even) => 짝수 번째 모든 태그에 스타일 적용
    • :nth-child(odd) => 홀수 번째 모든 태그에 스타일 적용
    • :nth-child(숫자) => 숫자번째 자식 태그에 스타일 적용
    • :first-child(숫자) => 형제관계에서 첫번째로 등장하는 태그에 스타일 적용
    • :last-child(숫자) => 형제관계에서 마지막으로 등장하는 태그에 스타일 적용
